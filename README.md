# Next Finish Carpentry - Modern Cedar Mailbox Landing Page

Landing page profissional e otimizada para conversão para venda de caixas de correio personalizadas de cedro moderno em Columbus, Ohio.

## 🚀 Features

- Design moderno e premium (Tailwind CSS)
- Totalmente responsiva (mobile-first)
- Formulário de cotação funcional com modal de sucesso
- FAQ interativo
- Múltiplos CTAs com telefone
- Pronto para deploy no Vercel ou Netlify

## 📁 Estrutura do Projeto

```
next-finish-cedar-mailbox-landing/
├── index.html          # Landing page completa
├── README.md
└── assets/
    └── mailbox-hero.jpg   # Adicione a imagem do flyer aqui
```

## 🚀 Como Publicar (Deploy Grátis)

### Opção 1: Vercel (Recomendado)
1. Acesse [vercel.com](https://vercel.com)
2. Faça login com GitHub
3. Clique em **New Project** > Importe este repositório
4. Deploy automático em segundos

### Opção 2: Netlify
1. Acesse [netlify.com](https://netlify.com)
2. Arraste a pasta do projeto ou conecte o GitHub
